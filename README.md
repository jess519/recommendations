# Single-Solver Prototype

Minimal React + Tailwind prototype for rapid iteration. Not for production.

## Run

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## Build (optional)

```bash
npm run build
npm run preview
```

## Stack

- **Vite** — fast dev server and HMR
- **React 18** — UI
- **Tailwind CSS v4** — styling via `@tailwindcss/vite`

Edit `src/App.jsx` and `src/index.css` to prototype. Add components under `src/` as needed.
